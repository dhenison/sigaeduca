# DESIGN.md --- SIGA EDUCA

## Identidade Visual

O SIGA EDUCA deve transmitir modernidade, confiança, acessibilidade e
organização.

### Estilo

-   SaaS moderno
-   Interface limpa
-   Cards com cantos arredondados (16px)
-   Sombras suaves
-   Layout responsivo (Desktop, Tablet e Mobile)

## Paleta de Cores

### Primária

-   Verde Principal: `#2EAF62`
-   Verde Claro: `#7EDC8A`

### Secundárias

-   Azul: `#3B82F6`
-   Amarelo: `#FBBF24`

### Neutras

-   Fundo: `#F7FAF8`
-   Cards: `#FFFFFF`
-   Texto Principal: `#1F2937`
-   Texto Secundário: `#6B7280`
-   Bordas: `#E5E7EB`

## Componentes

### Menu Lateral

-   Fundo branco
-   Ícones verdes
-   Item ativo com fundo verde claro

### Dashboard

Cards grandes com indicadores: - Alunos - Turmas - Frequência -
Ocorrências - Relatórios

Gráficos utilizando: - Verde - Azul - Amarelo

## UX

-   Navegação simples
-   Poucos cliques
-   Alta legibilidade
-   Ícones consistentes
-   Responsivo
